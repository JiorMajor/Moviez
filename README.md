# Moviez
Yet another movies app with the use of The Movie DB api


#### Features
- Browse Movies 
- Sort Movies by New Release and Popularity 
- View Details
- Buy ticker for the show from <a href="www.cathaycineplexes.com.sg">Cathay Web</a>.
